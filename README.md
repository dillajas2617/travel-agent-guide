- learn with simple comic panels

## 🪟 Install on Windows

If you downloaded an `.exe` or `.msi` file:

1. double-click the file
2. if Windows asks for permission, choose Yes
3. follow the setup steps on the screen
4. choose the install folder or keep the default one
5. finish the setup
6. open the app from the Start menu or desktop icon

If you downloaded a zipped file:

1. right-click the file
2. choose Extract All
3. pick a folder
4. open the extracted folder
5. run the main app file inside it

If Windows shows a SmartScreen message, check that you downloaded the file from the release page, then continue if the file name matches the latest release.
